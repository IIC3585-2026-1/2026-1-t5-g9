# MyUnits Svelte

Version Svelte de MyUnits, convertidor de unidades originalmente implementado en Vue 3 + Vite.

## Comandos

Instalar dependencias:

```bash
npm install
```

Levantar servidor de desarrollo:

```bash
npm run dev
```

Generar build de produccion:

```bash
npm run build
```

Previsualizar el build:

```bash
npm run preview
```
